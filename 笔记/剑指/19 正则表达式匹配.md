# 正则表达式匹配

题目 https://leetcode.cn/problems/zheng-ze-biao-da-shi-pi-pei-lcof/

## 思路：

两个字符串`pattern`（用于匹配）、`s`（被匹配的字符串）

两个指针`i`（指向`pattern`）、`j`（指向`s`）

1.   `pattern[i+1] != '*'`
     1.   `(s[j]!='\0' && pattern[i] == '.')|| pattern[i] == s[j] `（匹配成功）：`++i;++j`
     2.   不匹配：`return false`
2.   `pattern[i+1] == '*'`
     1.   ``(s[j]!='\0' && pattern[i] == '.')|| pattern[i] == s[j] `（当前字符匹配）：
          1.   `i+=2;`
          2.   `++j;`
          3.   `i+=2;++j;`
     2.   当前字符不匹配：`i+=2;`
